# Notepad

Обычный блокнот на Ruby c SQLite3.

# Как стартовать блокнот?

1. Для установки зависимостей 

```
bundle install
```
2. Создать заметку

```
ruby new_post.rb
```

3. Вывести все посты

```
ruby read.rb
```

# Посмотреть заметки

Таблицу SQLite можно открыть через программу DBeaver или DB Browser в файле notepad.sqlite.
